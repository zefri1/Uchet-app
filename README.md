# Учет недвижимости (УК Учет)

Простая и удобная программа для учета объектов недвижимости, арендаторов, выставления коммунальных счетов и распределения расходов. Разработано для использования внутри управляющей компании.

## Запуск приложения

Приложение кроссплатформенное и может быть запущено на Windows или macOS.

### Для Windows (простой способ)
Просто скачайте скомпилированный `.exe` файл из раздела [Releases](https://github.com/zefri1/Uchet-app/releases), запустите его и программа откроется в вашем браузере. Установка Python не требуется!

### Для macOS
Скачайте `.zip` архив с Mac-версией из раздела [Releases](https://github.com/zefri1/Uchet-app/releases), распакуйте его и запустите приложение `UK_Uchet_Mac.app`.
*Примечание: при первом запуске может потребоваться открыть программу через правый клик -> "Открыть", так как приложение не подписано сертификатом разработчика Apple.*

## Запуск из исходного кода (для разработчиков)

Если вы хотите запустить проект напрямую из исходников или внести изменения:

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите сервер:
   ```bash
   uvicorn app.main:app --reload
   ```
3. Откройте в браузере `http://127.0.0.1:8000/v2/`

### Демо-данные
Если база данных пустая, вы можете заполнить её тестовыми данными для проверки работы логики:
```bash
python seed_demo.py
```

## Сборка исполняемых файлов

Сборка осуществляется с помощью `PyInstaller`.
Для Windows:
```bash
pyinstaller УК_Учет_Win7_x64_v2.spec --clean -y
```

Для macOS настроена автоматическая сборка через GitHub Actions (см. `.github/workflows/build-mac.yml`).
