# Учет недвижимости

Локальное приложение для учета объектов, арендаторов, коммунальных счетов и распределения расходов.

## Самый простой вариант для сотрудника

1. Откройте папку проекта.
2. Дважды нажмите [УСТАНОВИТЬ_ПРИЛОЖЕНИЕ.bat](C:/Users/Alexey/Documents/ук/УСТАНОВИТЬ_ПРИЛОЖЕНИЕ.bat).
3. Дождитесь сообщения, что установка завершена.
4. После этого для обычной работы всегда запускайте [ЗАПУСТИТЬ_ПРИЛОЖЕНИЕ.bat](C:/Users/Alexey/Documents/ук/ЗАПУСТИТЬ_ПРИЛОЖЕНИЕ.bat).

Приложение откроется в браузере по адресу `http://127.0.0.1:8000`.

## Дополнительно

- Если хотите загрузить тестовые данные, запустите [ЗАПОЛНИТЬ_ДЕМО_ДАННЫМИ.bat](C:/Users/Alexey/Documents/ук/ЗАПОЛНИТЬ_ДЕМО_ДАННЫМИ.bat).
- Если Windows спросит про запуск файла, подтвердите запуск.
- Если на компьютере нет Python, установщик сам подскажет, что делать.

## Для администратора

- Код приложения: [app/main.py](C:/Users/Alexey/Documents/ук/app/main.py)
- Запуск вручную: `python run.py`
- Тестовые данные: `python seed_demo.py`
