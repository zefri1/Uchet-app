# План реализации нового интерфейса (Версия 2)

Основная цель: создать новый, упрощенный пользовательский интерфейс (V2), полностью сохранив старый (V1). Это позволит плавно протестировать новый подход, всегда имея возможность вернуться к старому.

## Фаза 1. Подготовка инфраструктуры (V2)
**Задача:** Создать независимую песочницу для нового интерфейса.

1. **Новые маршруты (Endpoints):**
   - Все новые страницы будут иметь префикс `/v2/`.
   - В `main.py` мы добавим новые обработчики (например, `@app.get("/v2/billing")`), которые будут рендерить новые шаблоны.
   - Существующие маршруты (например, `/billing`) останутся нетронутыми.

2. **Новые шаблоны:**
   - Создадим папку `app/templates/v2/`.
   - Создадим новый `base_v2.html` для V2 (с обновленным, сокращенным сайдбаром).
   - В старом сайдбаре (V1) добавим яркую кнопку: **«Перейти в новую версию 🚀»**.
   - В новом сайдбаре (V2) добавим кнопку: **«Вернуться в старую версию 🔙»**.

## Фаза 2. Справочники (Объекты, Арендаторы, Размещения)
**Задача:** Свести 3 страницы в одну удобную.

1. **Страница `/v2/directory`:**
   - Создаем шаблон `directory.html`.
   - Делаем 3 вкладки переключения (через JavaScript): «Объекты», «Арендаторы», «Размещение».
   - В форме Размещения выводим поле «Тариф аренды за 1 м²».

2. **Обработка данных:**
   - POST-запросы будут отправляться на старые API, но после успеха будут перенаправлять пользователя обратно на `/v2/directory`. Для этого добавим параметр `?redirect_to=/v2/directory` в формы.

## Фаза 3. Ежемесячный расчёт (Мастер / Единая страница)
**Задача:** Превратить процесс в пошаговый экран без хаоса таблиц.

1. **Страница `/v2/billing`:**
   - **Шаг 1: Период.** Страница показывает текущий активный период или предлагает создать новый (одна кнопка «Начать новый месяц», сама ставит даты).
   - **Шаг 2: Ввод счетов.** Выводим компактную таблицу счетов. Скрываем сложные поля («Режим распределения», «База площади») под кнопку «⚙ Дополнительно».
   - **Шаг 3: Расчёт и Итоги.** Одна кнопка «Рассчитать месяц». Появляется понятная сводка (сальдо/баланс).
   - **Шаг 4: Документы.** Большие кнопки скачивания счетов, актов и реестра.
